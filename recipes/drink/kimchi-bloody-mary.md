# Kimchi Bloody Mary

> [!NOTE] Origin
> Page 12 cocktail note from the Mockingbird Bar spread.

## Ingredients

*Base servings: 1 cocktail*

- [ ] `4 ounces` — kimchi base
- [ ] `2 ounces` — vodka
- [ ] `1/2 ounce` — ginger
- [ ] `1/2 ounce` — lemon juice
- [ ] `1/4 ounce` — spicy sauce, optional
- [ ] `garnish` — kimchi, gyoza chip, straw, chopsticks [ESTIMATED]

## Instructions

1. **Build over ice.** Add the kimchi base, vodka, ginger, lemon, and optional hot sauce to a glass over ice.
2. **Stir.** Mix gently so the drink stays savory and slightly briny.
3. **Garnish.** Finish with the playful garnish set just before service.

## Notes & Variations

### Cook's Notes
- The garnish reads like a bar-dinner flex, so keep the base balanced and the garnish theatrical.

### Make-Ahead / Storage
- Mix the liquid base ahead, but add garnish at the last moment.

### Scaling
- Multiply the liquid base, then garnish each drink individually.
